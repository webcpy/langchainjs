# 模块

LangChain 提供了以下模块的标准、可扩展接口和外部集成，这些模块按照从简单到复杂的顺序列出：

#### [模型 I/O](/docs/modules/model_io/)

与语言模型进行接口交互

#### [数据连接](/docs/modules/data_connection/)

与应用程序特定数据进行接口交互

#### [链](/docs/modules/chains/)

构建调用序列

#### [代理](/docs/modules/agents/)

让链根据高级指令选择使用哪些工具

#### [内存](/docs/modules/memory/)

在链的多次运行之间保持应用程序状态

#### [回调](/docs/modules/callbacks/)

记录和流式传输任何链的中间步骤

#### [实验性](/docs/modules/experimental/)

抽象尚未完全确定的实验性模块
