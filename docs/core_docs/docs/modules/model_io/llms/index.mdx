# 大型语言模型（LLMs）

大型语言模型（LLMs）是LangChain的核心组件。
LangChain不提供自己的LLMs，而是提供了一个标准接口，用于与许多不同的LLMs进行交互。具体而言，这个接口接收一个字符串作为输入，并返回一个字符串。

有许多LLM提供商（如OpenAI、Cohere、Hugging Face等），-`LLM`类的设计旨在为所有这些提供商提供一个标准接口。
