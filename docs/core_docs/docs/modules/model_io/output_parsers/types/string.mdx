# 字符串输出解析器

`StringOutputParser` 将语言模型的输出（可以是整个响应或流式传输）转换为字符串。这对于标准化聊天模型和LLM输出非常有用。

此输出解析器可以作为转换流，与模型的流式响应块一起工作。
