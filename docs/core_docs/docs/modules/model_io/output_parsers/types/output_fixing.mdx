# 自动修复解析器

这个输出解析器包装了另一个输出解析器，当第一个解析器失败时，它会调用另一个LLM来修复任何错误。

但我们不仅仅可以抛出错误。具体来说，我们可以将格式错误的输出以及格式化指令传递给模型，并要求其修复它。
