# 输出解析器

输出解析器负责接收LLM的输出并将其转换为更合适的格式。当您要求LLM生成任何形式的结构化数据时，这非常有用。

除了拥有大量不同类型的输出解析器之外，LangChain输出解析器的一个显著优势是其中许多支持流式处理。
