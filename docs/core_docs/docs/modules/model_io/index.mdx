# 模型I/O

任何语言模型应用的核心元素都是模型。LangChain为您提供了与任何语言模型进行交互的构建模块。

![model_io_diagram](/img/model_io.jpg)


## [概念指南](/docs/modules/model_io/concepts)

消息、提示、LLM与ChatModels的区别以及输出解析器的概念性解释。在开始之前，您应该阅读[这部分](/docs/modules/model_io/concepts)。

## [快速开始](/docs/modules/model_io/quick_start)

涵盖了开始使用不同类型模型的基础知识。如果你想了解功能概述，应该浏览[这部分](/docs/modules/model_io/quick_start)。

## [提示](/docs/modules/model_io/prompts)

[这部分](/docs/modules/model_io/prompts)深入探讨了不同类型的提示模板及其使用方法。

## [LLMs](/docs/modules/model_io/llms)

[这部分](/docs/modules/model_io/llms)涵盖了与LLM类相关的功能。LLM是一种模型，它接受文本字符串作为输入，并返回文本字符串。


## [ChatModels](/docs/modules/model_io/chat)

[这部分](/docs/modules/model_io/chat)涵盖了与ChatModel类相关的功能。ChatModel是一种模型，它接受消息列表作为输入，并返回一条消息。


## [输出解析器](/docs/modules/model_io/output_parsers)

输出解析器负责将LLM和ChatModel的输出转换为更结构化的数据。[这部分](/docs/modules/model_io/output_parsers)介绍了不同类型的输出解析器。
