# 组合

本文介绍了如何将多个提示组合在一起。当您想要重用提示的部分时，这可能会很有用。这可以通过PipelinePrompt来完成。PipelinePrompt包括两个主要部分

- Final prompt：这是返回的最终提示
- Pipeline prompts：这是一个由字符串名称和提示模板组成的元组列表。每个提示模板将被格式化，然后作为具有相同名称的变量传递给未来的提示模板。
