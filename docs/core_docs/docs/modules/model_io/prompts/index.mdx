# 提示词

语言模型的提示词是由用户提供的一组一个语言模型的提示是用户提供的一组指令或输入，用于指导模型的响应，帮助它理解上下文并生成相关和连贯的基于语言的输出，
例如回答问题、完成句子或参与对话。
