## Deeplearning.ai

我们与 [Deeplearning.ai](https://deeplearning.ai) 和 [Andrew Ng](https://en.wikipedia.org/wiki/Andrew_Ng) 合作推出了一个 LangChain.js 短课程。

它涵盖了 LCEL 和其他可以组合起来构建更复杂链的构建块，以及关于加载数据以进行检索增强生成（RAG）的基础知识。免费尝试如下：

- [使用 LangChain.js 构建 LLM 应用](https://www.deeplearning.ai/short-courses/build-llm-apps-with-langchain-js)

## Scrimba 互动指南

[Scrimba](https://scrimba.com) 是一个代码学习平台，允许您在观看视频教程的同时交互式地编辑和运行代码。

我们与 Scrimba 合作推出了课程材料（称为“scrims”），教授使用 LangChain.js 构建的基础知识 - 在下面查看它们，并随着它们的推出回来查看更多！

### 学习 LangChain.js

- [在 Scrimba 上学习 LangChain.js](https://scrimba.com/learn/langchain)

一个完整的端到端课程，指导如何构建一个可以回答关于提供的文档的问题的聊天机器人。是学习 LangChain 的绝佳入门，也是学习如何使用 LangChain 表达语言（LCEL）原语进行检索的绝佳第一个项目！
