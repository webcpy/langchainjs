# 表格问答

大量的数据和信息存储在表格数据中，无论是 csv 文件、Excel 表格还是 SQL 数据库表。
本页面涵盖了 LangChain 中用于处理此类数据格式的所有可用资源。

## 链（Chains）

如果您刚开始接触，并且有相对较小/简单的表格数据，您应该从链开始。
链是一系列预定义的步骤，因此它们非常适合初学者，因为它们可以为您提供更多的控制权，并让您更好地理解正在发生的事情。

- [SQL 数据库链](../modules/chains/popular/sqlite)

## 代理（Agents）

代理更为复杂，涉及对 LLM 进行多次查询以了解要做什么。
代理的缺点是您的控制权较少。优点是它们更加强大，
这允许您在更大的数据库和更复杂的模式上使用它们。

- [SQL 代理](/docs/integrations/toolkits/sql)
