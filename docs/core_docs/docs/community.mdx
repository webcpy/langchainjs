# 🦜 为 LangChain 做贡献

LangChain 是由超过 5,000+ 次的贡献和 1,500+ 贡献者共同创造的，我们**仍然**有很多事情可以一起做。以下是一些参与方式：

- **[开启一个 pull request](https://github.com/langchain-ai/langchainjs/issues)：** 我们会感激所有形式的贡献——新功能、基础设施改进、更好的文档、错误修复等。如果您有改进或想法，我们很愿意与您一起工作。
- **[阅读我们的贡献者指南：](https://github.com/langchain-ai/langchainjs/blob/main/CONTRIBUTING.md)** 我们要求贡献者遵循 ["fork and pull request"](https://docs.github.com/en/get-started/quickstart/contributing-to-projects) 工作流程，在提交之前运行一些本地格式化、代码检查和测试，并遵循某些文档和测试约定。
- **成为专家：** 我们的专家通过在 Discord 中回答产品问题来帮助社区。如果您愿意扮演这个角色，我们将非常感激！（我们还有一些仅限专家的特别福利，我们可以告诉您更多）。发送电子邮件至 <hello@langchain.dev> 介绍自己，我们将从这里开始！
- **与 LangChain 集成：** 如果您的产品与 LangChain 集成——或者有这个愿望——我们希望帮助确保您和最终用户的体验尽可能顺利。发送电子邮件至 <hello@langchain.dev> 告诉我们您正在做什么。
  - **成为集成维护者：** 与我们的团队合作，确保您的集成保持最新，并在我们的 Discord 中直接与用户交谈（并回答他们的问题）。如果您想探索这个角色，请发送电子邮件至 <hello@langchain.dev> 介绍自己。

# 🌍 聚会、活动和黑客松

我们最喜欢在 AI 领域工作的一点就是人们对共同构建的热情。我们希望为您使这一过程尽可能简单和有影响力！

- **寻找聚会、黑客松或网络研讨会：** 您可以在我们的 [全球活动日历](https://mirror-feeling-d80.notion.site/0bc81da76a184297b86ca8fc782ee9a3?v=0d80342540df465396546976a50cfb3f) 上找到适合您的活动。
  - **向我们的日历提交事件：** 发送电子邮件至 <events@langchain.dev> 并附上您的事件页面链接！我们还可以帮助您通过我们的本地社区传播这个消息。
- **举办聚会：** 如果您想将一群构建者聚集在一起，我们愿意提供帮助！我们可以在我们的活动日历/Twitter 上宣传您的事件，通过 Discord 与我们的本地社区分享，发送赠品，或者可能为您找到赞助商。发送电子邮件至 <events@langchain.dev> 告诉我们您的事件！
- **成为聚会赞助商：** 我们经常听到有构建者团体想要聚集，但在某些方面受到限制（如场地、预算、奖品等）。如果您愿意提供帮助，请发送电子邮件至 <events@langchain.dev>，我们可以分享更多关于它如何运作的信息！
- **在活动中发言：** 聚会主持人总是在寻找优秀的演讲者、展示者和小组讨论成员。如果您想在活动中担任这些角色，请发送电子邮件至 <hello@langchain.dev> 并提供更多关于您自己、您想谈论的内容以及您所在的城市的信息，我们将尝试将您与即将进行的活动匹配！
- **告诉我们您的 LLM 社区：** 如果您主持或参与一个社区，并且欢迎来自 LangChain 和/或我们团队的支持，请发送电子邮件至 <hello@langchain.dev> 并告诉我们我们可以如何提供帮助。

# 📣 帮助我们放大您的工作

如果您正在从事让您感到自豪的工作，并且认为LangChain社区会从了解它中受益，我们愿意帮助您展示。

- **发布您的工作并提及我们：** 我们喜欢在Twitter上闲逛，看看这个领域的人们在谈论和做什么。如果您标记[@langchainai](https://twitter.com/LangChainAI)，我们几乎肯定能看到它，并可以给您一些关注。
- **在我们的博客上发表文章：** 如果您正在撰写关于使用LangChain构建的经验，我们很乐意在博客上发布（或交叉发布）！请将您的文章草稿发送至<hello@langchain.dev>！或者提出您想写的主题想法。
- **将您的产品加入我们的[集成中心](https://integrations.langchain.com/)：** 许多开发者利用我们与其他产品的无缝集成，并来到我们的集成中心了解这些产品。如果您想将您的产品加入其中，请告诉我们关于它的情况（以及它与LangChain如何协同工作）至<hello@langchain.dev>。
