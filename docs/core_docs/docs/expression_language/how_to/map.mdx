## 操作输出/输入

映射在将一个 Runnable 的输出转换为序列中下一个 Runnable 的输入格式时非常有用。

注意，在 `RunnableSequence.from()` 调用中的对象会自动被强制转换为 runnable 映射。对象的所有键
必须有值是 runnables 或者可以被强制转换为 runnables（函数到 `RunnableLambda` 或者对象到 `RunnableMap`）。
这种强制转换也会在通过 `.pipe()` 方法组合链时发生。
