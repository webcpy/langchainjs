# 取消请求

您可以通过绑定一个 `signal` 来取消一个 LCEL 请求。

```bash npm2yarn
npm install @langchain/openai
```

```typescript
import { OpenAI } from '@langchain/openai';
import { CancelController } from '@langchain/core';

const controller = new CancelController();
const openai = new OpenAI({ cancelToken: controller.signal });

const run = async () => {
  const response = await openai.call(
    'completions.create',
    'The quick brown fox jumps'
  );
  console.log(response);

  // 取消请求
  controller.abort();
};

run().catch(console.error);
```

请注意，这仅在基础提供者暴露了该选项时才会取消输出请求。如果可能，LangChain将取消基础请求，否则将取消响应的处理。
