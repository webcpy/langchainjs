# LangChain Expression Language (LCEL)

LangChain Expression Language（LCEL）是一种声明式的方法，可以轻松地将链条组合在一起。使用这种方式构建的任何链条都会自动拥有完整的同步、异步和流式支持。LCEL旨在提供一种简单而强大的方式来定义文本处理流程，使用户能够以声明性的方式组合各种处理步骤，同时获得灵活的同步和异步处理能力，以及流式数据处理的支持。

如果你正在寻找一个好的入门指南，请查看 [示例集](/docs/expression_language/cookbook) ——它按从简单到复杂的顺序展示了各种表达式语言部分。
